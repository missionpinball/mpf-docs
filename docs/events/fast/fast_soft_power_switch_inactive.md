# fast_soft_power_switch_inactive


--8<-- "event_deeper.md"

!!! info ""

    Event available in MPF 0.81

Event is posted by the FAST Neuron's soft power switch when released.

This is not a normal switch event, and is instead reported through
the watchdog command on the NET subprocessor.
