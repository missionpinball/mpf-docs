# multiball_(name)\_restarted


--8<-- "event.md"

Event is posted by [multiballs:](../config/multiballs.md)

The multiball called (name) is has restarted during the restart grace period.
This event is posted when the multiball add-a-ball handler is called, causing
a successful restart.
