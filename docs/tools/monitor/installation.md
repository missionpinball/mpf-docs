# Installing MPF Monitor

As of January 2026, the current stable version of MPF Monitor is `0.57.2`.

You can install it via `pip` (or `pip3`, or however you installed MPF).

``` doscon
$ pip install mpf-monitor
```

Old versions of monitor can be found on [Github](https://github.com/missionpinball/mpf-monitor/tags)
and [Pypi](https://pypi.org/project/mpf-monitor/#history).
