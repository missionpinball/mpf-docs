Using MPF with existing pinball machines (Williams, Stern, Gottlieb, etc.)
==========================================================================

MPF supports all kinds of pinball machines. In this section, we highlight
how to connect and configure existing machines with MPF.

* :doc:`Williams, Bally, Midway WPC, WPC-S, WPC-95 <wpc>`
* :doc:`Williams, Bally System 11 <system11>`
* :doc:`Gottlieb System 1 <gottlieb_system1>`
* :doc:`Gottlieb System 80 <gottlieb_system80>`
* :doc:`Data East <data_east>`
* :doc:`Stern Whitestar <whitestar>`
* :doc:`Stern SAM <sam>`
* :doc:`Stern SPIKE / SPIKE 2 <spike>`
* :doc:`Pinball 2000 <pinball2000>`

Please let us know if you want to connect any other machine.
